## 功能
- 支持图层模式， 使用 source  / reference  命名图层组， 可以得到类似chat中发送两张图的效果
- 文生图，图生图
- 选区模式，可只把选区区域发送给ai进行生图
- Prompt预设
- 同时处理多生图任务功能

## Features
- Supports layer modes: Name layer groups `source` / `reference` to achieve an effect similar to sending two images in a chat.
- Text-to-Image, Image-to-Image generation.
- Selection mode: Only send the selected area to AI for image generation.
- Prompt presets.
- Ability to process multiple image generation tasks simultaneously.

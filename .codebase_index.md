# PSBananaUXP 代码库索引

更新时间: 2026-02-27

## 1. 项目概览
- 类型: Photoshop UXP 插件（AI 生图）
- 插件入口: `index.html` -> `main.js`
- 主机配置: `manifest.json`（`host.app=PS`，`minVersion=23.0.0`，`apiVersion=2`）
- 核心能力: 文生图、图生图、多图模式（source/reference）、选区生图、Provider 多适配

## 4. 入口与调用链
1. `index.html` 加载 `main.js`
2. `main.js` 初始化 `SettingsManager/ProviderManager/PresetManager/FileManager/ImageGenerator`
3. 触发生图时：
   - 收集 UI 参数与 Photoshop 上下文
   - `PSOperations` 导出输入图/选区信息
   - `ImageGenerator` 按 Provider 构造请求并调用远端 API
   - `FileManager` 保存返回图像
   - `PSOperations` 将结果导入并落到文档图层

## 5. 与 AI Provider 相关的关键点
- Provider 识别与适配入口: `api_providers.js#getProviderConfig`
- 请求主体拼装: `image_generator.js#_buildPayload`（按 providerType 分流）
- 返回解析与图片提取: `image_generator.js#_processResponse`（不同返回格式分支）
- ComfyUI 模式模板: `workflow_templates.js`

## 6. 运行时数据（非源码）
`Profiles/` 目录用于本地缓存和调试，不建议作为功能代码阅读入口。
- `Profiles/settings.json`: 全局设置
- `Profiles/providers.json`: Provider 列表与连接配置
- `Profiles/presets.json`: Prompt 预设
- `Profiles/Logs/`: 调试日志、payload、response

## 8. 快速定位建议
- 改 UI/交互: 先看 `index.html`，再看 `main.js`
- 改 Photoshop 文档/图层逻辑: `ps_operations.js`
- 改 API 对接或新增 Provider: `api_providers.js` + `image_generator.js`
- 改本地缓存/日志/文件导入导出: `file_manager.js`
- 改配置与预设: `settings_manager.js` / `presets_manager.js`
